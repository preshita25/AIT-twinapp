# AI Twin Learning Platform

An interactive learning platform that visualizes a student’s academic journey,
gamifies learning through quizzes and badges, and enables teachers to share
insights with parents.

## Features

- Grade 1–10 learning timeline
- Gamified quizzes with levels and badges
- Teacher → Parent insight sharing (demo)
- Modern UI built with React, TypeScript, and Tailwind CSS

## Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS
- shadcn/ui

## Status

Hackathon prototype (frontend complete, backend-ready).
